# Firstname Lastname, PhD - Personal Academic Website

This is a personal site for showcasing academic research and sport psychology services.

## Built with
- React
- Tailwind CSS
- Vite
- Deployed with Vercel

## Deploying
1. Push this project to GitHub.
2. Connect the repo to your Vercel account.
3. Vercel will auto-detect the Vite app and deploy it.

You're done!
